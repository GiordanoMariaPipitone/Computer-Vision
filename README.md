# Computer Vision

Raccolta di progetti di Computer Vision e Deep Learning: classificazione di immagini, detection e rappresentazioni oltre l'RGB.

## Progetti

| Progetto | Descrizione | File |
|---|---|---|
| Weather Classification | Classificazione di immagini di condizioni meteo | `Weather_Classification.ipynb` |
| Brain Tumor (TensorFlow) | Classificazione di tumori cerebrali da immagini MRI | `[Tensorflow]Brain_Tumor.ipynb` |
| Rice Grain (TensorFlow) | Classificazione di varietà di chicchi di riso | `[Tensorflow]Rice_Grain.ipynb` |
| Plant Diseases Detection | Classificazione di malattie delle piante da immagini fogliari (New Plant Diseases Dataset) | `Plant_Diseases_Detection.ipynb` |
| Deepfake Detection | Rilevamento di immagini/volti deepfake | `Deepfake_Detection.ipynb` |
| Lingua dei Segni | Riconoscimento del dataset della lingua dei segni | `Lingua_dei_segni.ipynb` |
| Oltre l'RGB | Imaging oltre lo spettro RGB: rappresentazioni iper-dimensionali per diagnostica e classificazione | [Oltre l'RGB/](<./Oltre l'RGB>) |

## Stack

Python · TensorFlow / Keras · PyTorch · NumPy · Matplotlib · Jupyter
